    Coercion

    In JavaScript, there are three components of the language that especially lend themselves to the creation of jargon:

    1. coercion,
    2. logical operators, &
    3. bitwise manipulations (pejoratively known as bit twiddling.)

    Coercion

    In JavaScript as in most other languages, coercion is the act of forcing an object or entity of one type into another. This is not to be confused with type conversion, which is the explicit transformation between types. In JavaScript, explicit type conversion would look like this:

    // => "1"
var a = (1).toString();
console.log(a);

// However, the number can also be implicitly coerced into a string this way:
// => "1"
var a = 1 + "";
console.log(a);
