    Difference between Single & Double Curly Braces

    In JavaScript (and JSX), {} and {{}} are used in different contexts, and here's the distinction:

    Success

    • In JSX: {} is used to embed JavaScript expressions into JSX.

      • Example: const name = 'John'; and <div>{name}</div> will render the value of name inside the <div>.

    • In JavaScript: {} is also used for defining objects, blocks, or destructuring.

      • In JSX: When you want to embed a variable or a JavaScript expression (like a number, string, function call, etc.) inside JSX, you wrap it with {}. For example:

        const greeting = 'Hello, World!';
return <h1>{greeting}</h1>;  // The value of greeting will be displayed inside the <h1>

    • In JSX: {{}} is commonly used when you're passing an object as a prop, especially for inline styles or for embedding an object inside JSX.

      • For inline styles, you need to use a double pair of curly braces:

        const style = { color: 'blue', fontSize: '16px' };
return <div style={{ color: 'blue', fontSize: '16px' }}>Styled Text</div>;
        • The first pair {} is for the JSX syntax (since it's in JSX, it expects expressions).
        • The second pair {{}} is the object itself that you want to pass (the inline styles are represented as an object).

      • Another use case is when you're passing an object to a function or a component prop:

        <Component prop={{ key: 'value' }} />

        This means you're passing an object { key: 'value' } to the prop of the Component.

    • {} (Single Curly Braces): Used to embed JavaScript expressions in JSX.
    • {{}} (Double Curly Braces): Used when you want to pass an object (like inline styles or any other object) inside JSX.

    import { useState, useCallback } from "react";
import { Layout } from "antd";
import { RightCircleOutlined, LeftCircleOutlined } from "@ant-design/icons";

import Header from "./Header";
import Sidebar from "./Sidebar";

const { Content } = Layout;

const MainLayout = ({ children }) => {
    const [collapsed, setCollapsed] = useState(false);

    // Using useCallback to memoize the function and prevent unnecessary re-renders
    const toggleSidebar = useCallback(() => {
        setCollapsed((prev) => !prev);
    }, []);

    const sidebarWidth = collapsed ? 80 : 240;
    const contentWidth = `calc(100% - ${sidebarWidth}px)`;

    return (
        <Layout>
            <Header toggleSidebar={toggleSidebar} /> {/* Pass toggleSidebar to Header */}
            <Layout style={{ display: "flex" }}>
                <Sidebar collapsed={collapsed} width={sidebarWidth} />
                <Content style={{ width: contentWidth, position: "relative" }}>
                    {collapsed ? (
                        <RightCircleOutlined
                            onClick={toggleSidebar}
                            aria-label="Expand Sidebar"
                        />
                    ) : (
                        <LeftCircleOutlined
                            onClick={toggleSidebar}
                            aria-label="Collapse Sidebar"
                        />
                    )}
                    {children}
                </Content>
            </Layout>
        </Layout>
    );
};

export default MainLayout;

    import { Layout } from "antd";
import Header from "./Header";
import Sidebar from "./Sidebar";
import { useState } from "react";
import { useMediaQuery } from "react-responsive";

const { Content } = Layout;

const MainLayout = ({ children }) => {
    // State for sidebar collapse
    const [collapsed, setCollapsed] = useState(false);

    // Set sidebar width based on collapse state
    const sidebarWidth = collapsed ? 80 : 240;

    // Use media query to check if the screen is small
    const isSmallScreen = useMediaQuery({ maxWidth: 768 });

    // Automatically collapse sidebar on small screens
    if (isSmallScreen && !collapsed) {
        setCollapsed(true);
    }

    // Calculate content width based on sidebar state
    const contentWidth = `calc(100% - ${sidebarWidth}px)`;

    return (
        <Layout style={{ minHeight: "100vh" }}>
            <Header collapsed={collapsed} setCollapsed={setCollapsed} />
            <Layout style={{ display: "flex" }}>
                <Sidebar collapsed={collapsed} width={sidebarWidth} />
                <Content style={{ width: contentWidth }}>{children}</Content>
            </Layout>
        </Layout>
    );
};

export default MainLayout;
