    Details

    1. Arithmetic Operators

    These operators perform mathematical operations like addition, subtraction, multiplication, division, etc.

    • + (Addition)
    • - (Subtraction)
    • * (Multiplication)
    • / (Division)
    • % (Remainder, Modulo)
    • ++ (Increment)

    • -- (Decrement)

      let a = 5;
let b = 2;
console.log(a + b); // 7
console.log(a - b); // 3
console.log(a * b); // 10
console.log(a / b); // 2.5
console.log(a % b); // 1

    2. Assignment Operators

    These are used to assign values to variables.

    • = (Assignment)
    • += (Add and assign)
    • -= (Subtract and assign)
    • *= (Multiply and assign)
    • /= (Divide and assign)

    • %= (Modulo and assign)

      let x = 5;
x += 3; // x = x + 3 (x becomes 8)

    3. Comparison Operators

    These are used to compare two values and return a Boolean result (true or false).

    • == (Equal to)
    • === (Strict equal to: checks both value and type)
    • != (Not equal to)
    • !== (Strict not equal to)
    • > (Greater than)
    • < (Less than)
    • >= (Greater than or equal to)

    • <= (Less than or equal to)

      console.log(5 == "5"); // true (due to type coercion)
console.log(5 === "5"); // false (strict comparison)

    4. Logical Operators

    These are used to perform logical operations, typically with boolean values.

    • && (Logical AND)
    • || (Logical OR)

    • ! (Logical NOT)

      let a = true;
let b = false;
console.log(a && b); // false
console.log(a || b); // true
console.log(!a); // false

    5. Bitwise Operators

    These perform operations on the binary representations of numbers.

    • & (AND)
    • | (OR)
    • ^ (XOR)
    • ~ (NOT)
    • << (Left shift)
    • >> (Right shift)

    • >>> (Unsigned right shift)

      let a = 5; // 0101 in binary
let b = 3; // 0011 in binary
console.log(a & b); // 1 (0001 in binary)

    6. Ternary (Conditional) Operator

    This is a shorthand for the if-else statement.

    • condition ? expr1 : expr2

      If the condition is true, expr1 is executed; otherwise, expr2 is executed.

      let age = 20;
let isAdult = age >= 18 ? "Yes" : "No"; // 'Yes'

    7. Type Operators

    These operators deal with data types.

    • typeof (Returns the type of a variable or expression)

    • instanceof (Checks if an object is an instance of a particular class)

      let num = 5;
console.log(typeof num); // 'number'

let str = "hello";
console.log(str instanceof String); // false

    8. Spread and Rest Operators

    These are used in function calls and object manipulations.

    • ... (Spread / Rest Operator)

      Example (Spread in array)
      let arr = [1, 2, 3];
let newArr = [...arr, 4, 5];
console.log(newArr); // [1, 2, 3, 4, 5]
      Example (Rest in function parameters)
      function sum(...numbers) {
  return numbers.reduce((acc, num) => acc + num, 0);
}
console.log(sum(1, 2, 3)); // 6

    9. Unary Operators

    These operators operate on a single operand.

    • + (Unary plus: converts the operand to a number)
    • - (Unary negation: negates the operand)
    • ++ (Increment: increases the operand by 1)
    • -- (Decrement: decreases the operand by 1)
    • typeof (Returns the type of the operand)

    • delete (Deletes a property of an object)

      let x = 5;
console.log(++x); // 6 (pre-increment)
console.log(x--); // 6 (post-decrement)

    10. Nullish Coalescing Operator (??)

    This returns the right-hand operand when the left-hand operand is null or undefined.

    let foo = null;
let bar = "Hello";
console.log(foo ?? bar); // 'Hello'

    11. Optional Chaining (?.)

    This allows you to access deeply nested properties without having to check if each reference in the chain is valid.

    let user = { name: "Alice", address: { city: "New York" } };
console.log(user?.address?.city); // 'New York'
console.log(user?.phone?.number); // undefined (does not throw an error)

    These are the primary categories of operators in JavaScript, each serving a unique purpose to manipulate data and perform calculations.
